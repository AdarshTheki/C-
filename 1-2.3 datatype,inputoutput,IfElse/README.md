## 1 - DataType:
There are Three type of DataType in C++ Programming. 

**Primitive DataTypes** : 
```c++
int, float, char, bool, double float, void, wchar_t. 
```
**Derived DataTypes** : 
```
Function(), Array[], Pointer(*), Reference(&).
```
**Abstract/User-Defined DataTypes** :
```c++
class(), struct(), union(), enum(), typedef() 
```
